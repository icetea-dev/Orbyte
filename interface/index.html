Controller App Setup

To use Slash Commands everywhere, you need a User-Installed Bot (Controller).

Controller Setup Guide

  1. Go to Discord Developer Portal.
  2. Click New Application -> Name it (e.g. "MyController").
  3. Go to Installation tab.
  4. Uncheck "Guild Install". Check "User Install".
  5. In "Install Link", select "None".
  6. Go to Bot tab.
  7. IMPORTANT: Uncheck "Public Bot".
  8. Click Reset Token -> Copy it (This is your Bot Token).
  9. Go to General Information -> Copy Application ID.
  10. Replace YOUR_ID in this link:
    https://discord.com/oauth2/authorize?client_id=YOUR_ID&integration_type=1&scope=applications.commands
  11. Open that link in your browser to Authorize the bot on your account.

Once authorized, paste the Bot Token here.
